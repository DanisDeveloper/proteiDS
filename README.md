## Инструкция по запуску
Ввести команды ниже в консоль
```
git clone https://github.com/DanisDeveloper/proteiDS.git
cd proteiDS
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install jupyterthemes
jt -t onedork -T -N -kl
jupyter notebook
```
